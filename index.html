Projects

Nand2Tetris

Starting with nothing but nand gates, I built a fully operational 16-bit virtual computer, constructing virtual computer chips using a basic HDL code.

Additionally, I wrote an assembler to handle the machine code and a VM translator to enable higher-level programming on the computer. This project was based on a lecture series.


Virtual Pet Shelter

Created shelters for different kinds of virtual pets: a robotic shelter for robotic cats and dogs, and an organic shelter for organic cats and dogs.

Each of the pets have a set of methods i.e. train, feed for organic pets, charge for robotic pets, play fetch for dogs, etc. This project showcases working knowledge of important features of Java and object-oriented programming such as encapsulation, inheritance, polymorphism, etc.
